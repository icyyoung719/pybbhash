# Python Version of BBHash
This repository contains a Python implementation of the BBHash minimal perfect hash function (MPHF) algorithm, compatible with the original C++ implementation.

See [BINARY_FORMAT.md](docs/BINARY_FORMAT.md) for details on the binary serialization format used by this library.
